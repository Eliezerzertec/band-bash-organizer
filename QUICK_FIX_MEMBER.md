# 🚀 CORREÇÃO PARA CRIAR MEMBRO - AÇÃO RÁPIDA

## ❌ Erro ao Criar Membro? Faça Isto:

### 3 Passos (2 minutos)

1. **Abra Supabase**
   - Acesse: https://app.supabase.com
   - Seu projeto → **SQL Editor**

2. **Execute o Script**
   - Clique em **New Query**
   - Copie arquivo: `fix_profiles_policies.sql`
   - Cole no editor
   - Clique em **Execute**

3. **Teste Novamente**
   - Volte para app
   - Members → Novo Membro
   - Preencha e clique Criar

✅ Pronto!
